# AI Tutor - Your Personal Study Assistant

An AI-powered tutoring platform that helps students learn using their **own course materials**. No more AI hallucinations or off-curriculum answers, this tutor stays strictly within your uploaded documents.

## Key Features

🎓 **Course Management** - Create courses and upload all your materials (PDF, Word, PowerPoint, images)

💬 **Interactive Chat** - Ask questions about your materials and get answers with citations

📖 **Document Viewer** - Read your materials side-by-side with an AI assistant

🧩 **Problem Solving** - Get step-by-step guidance on exercises using your course content

✅ **Self-Assessment** - Generate practice questions (MCQs and open-ended) to test your understanding

## The Cool Part

If the answer isn't in your uploaded materials, it tells you so. This means:
- No hallucinations
- Follows your teacher's specific methods
- Always shows you where it found the information
- Stays aligned with your curriculum

## Tech Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** FastAPI (Python)
- **AI:** Google Gemini
- **Database:** PostgreSQL + Pinecone (vector database)
- **File Support:** PDF, Word, PowerPoint, Images

## Getting Started

1. Set up the backend (see `backend/` directory)
2. Set up the frontend (see `frontend/` directory)
3. Configure your AI API keys
4. Start learning! 📚
